# BOOTSTRAP.md

Use this file when adapting an empty or existing folder into an Agentic Repository.

1. Inspect the current directory tree.
2. Preserve existing user files.
3. Create missing directories from the default schema.
4. Add or update `AGENTS.md`, `wiki/index.md`, `wiki/log.md`, and `work/inbox.md`.
5. Record bootstrap notes in `wiki/log.md`.

For non-empty repositories, propose a plan before changing files.
