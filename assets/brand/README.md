# OpenClaw Desktop Brand Assets

Generated by `node scripts/export-brand-assets.mjs`.

## Direction

OpenClaw Desktop uses the Crayfish Operator D2 identity with image2-generated chroma-key masters. OpenClaw Desktop Club uses the same mascot system for the community/site brand.

## Alpha Policy

Image2 generates high-quality artwork on a pure chroma background. The production package converts those chroma masters into transparent PNG masters, then exports light/dark transparent logos and semi-transparent panel variants.

Every production PNG that is intended to be transparent must be validated as sRGBA with transparent corners.
