# 设计系统

> UI 框架：Semi Design（字节跳动） / 待定
> 定位："精美易用"的 AI 控制台

## 设计哲学

1. **不干扰，不打断**
   - 信息密度适中，避免认知过载
   - 断线提示、状态变更使用温和的 Message 条
   - 3D 办公室作为可选视图，不强制

2. **可视化优先**
   - 数据指标用卡片和图表呈现
   - Agent 状态用颜色、图标、动画表达
   - 协作关系用拓扑图/热力图

3. **黑暗模式第一**
   - 桌面工具以黑暗模式为主
   - 明亮模式作为备选
