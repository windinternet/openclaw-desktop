# OpenClaw Desktop v0.1.2

本次发布说明按 `v0.1.1` 后到 release-notes workflow 修复提交的范围汇总，是一次较大的桌面端能力、品牌资产、3D 办公室、产物系统和工作台升级。

## Scope

- 起点：`v0.1.1` / `4f8828279dc7e7e3812a58b593527e7890699ea6`
- `v0.1.2` 发布 tag：`b02fcf72b4a166abc33f1d0d01ca3a416cac0d9a`
- 当前说明同步到：`6e9732e145fb552013f6bcadf1b2f65422c4141d`
- 覆盖 `113` 个提交，`318` 个文件变化，约 `26,516` 行新增与 `2,113` 行删除。
- 后续仅用于扩展本说明文件本身的 docs 提交不计入功能变更统计。

## Highlights

### Agentic Repository Workbench

- 新增 Agentic Repository 资源骨架，包含 outputs、plans、runs、schemas、sources、wiki、work 等目录与说明文件。
- 新增仓库绑定入口、Repository Gate、知识库视图、Workbench 视图、协议控制页和状态视图。
- 支持知识条目关系、仓库活动展示、Markdown 预览、工作台运行与输出镜像。
- 将旧产物命令兼容到新的 repository outputs 流程，并暴露 companion repository 命令基础。
- 打磨 Workbench 与会话/聊天的联动，恢复标签头布局，并将 Kanban 移到 Workbench 场景内。

### Artifact System

- 扩展产物类型：`link`、`app`、`file`、`audio`、`image`、`video`，并补齐 `GenerateParams`。
- 新增 AI 魔法创建产物提示词、创建入口、动态表单和弹窗流程。
- 产物列表与详情页支持类型差异化展示、类型专属操作按钮与字段展示。
- 同步 node commands、parser 和 artifact service 对新产物类型的支持，修正 HTML 强制要求问题。
- 补充中英文 i18n，并统一新建产物与魔法创建弹窗的挂载/显示策略，修复首次不显示等交互问题。

### Desktop Pet

- 新增桌宠独立窗口入口、Vite 多入口配置、Electron 主进程窗口管理、持久化、IPC handler 与 preload wiring。
- 新增 pet bridge、事件类型、事件总线，并可从 Gateway 连接回调触发桌宠事件。
- 新增桌宠渲染引擎、动画状态机、物理边界、气泡系统、语录和精灵管理。
- 新增 PetControl，并将控制入口接入设置页/侧边栏相关流程。
- 修复桌宠 race condition、状态损坏、生产构建、AI link、sprite 路径、IPC 清理、拖拽、气泡定位和 canvas 布局问题。

### 3D Office

- 新增 3D 办公室 playful blaster 互动玩法、规则、反馈和输入命中辅助。
- 补齐 office gameplay、scene interaction、audio、theme 等测试覆盖。
- 强化办公室场景交互：修正输入目标识别，防止倒下角色继续响应控制，提升 blaster 场景稳定性。
- 补充办公室玩法、材质/主题、龙虾运营室方向等设计与实施文档。

### Dashboard and Usage Insights

- 重设计 Dashboard quick start 区域，接入真实 Gateway usage metrics。
- 新增 Gateway 使用量汇总逻辑，修正 Dashboard Gateway summary。
- 新增 AntV 图表渲染、UsageTrendChart、DashboardVisualCharts 与图表格式化逻辑。
- 打磨使用量图表坐标轴、容器填充和视觉呈现。

### Navigation, Sidebar, and Sessions

- 新增主导航模型、Hub 页面、路由注册和导航文案。
- 重构侧边栏导航，支持更宽的 300px 布局、紧凑导航、分组偏好和加粗分组标签。
- 保留 scheduled tasks 在控制中心内，补齐导航 locale 路径测试。
- 会话列表标题新增 Tooltip 展示完整名称。
- 修复会话详情默认面板、标签唯一性、侧边栏宽度冲突和合并冲突。
- 为侧边栏顶部用户头像与底部账号头像增加 fallback 背景，避免头像加载失败时出现空白。
- 暴露 session tool 与 reasoning display 相关设置，并修复桌面端 console warning。

### Brand Assets

- 新增 OpenClaw 小龙虾吉祥物 app icons，并调整图标背景亮度。
- 新增完整品牌资产包：OpenClaw Desktop logo、OpenClaw Desktop Club logo/badge、横版 logo、透明版本、亮色/暗色 panel 版本、app icon 多尺寸输出。
- 保存 AI master、透明 master、chroma master、source HTML/SVG、raw AI render 等源文件，方便后续继续迭代。
- 更新 `build/icons`、`src/assets/apple-touch-icon.png` 与品牌资产 manifest，并补充品牌资产测试。

### Platform, Gateway, and Installation Reliability

- 新增主进程与渲染进程文件日志系统。
- Companion 安装提示支持“不再提醒”。
- 添加实例时 Token 改为可选。
- OpenClaw 检测改用 login shell，改善 AppImage 环境兼容性。
- 适配 Windows 平台命令执行、终端启动与安装流程。
- 强化 Gateway 自动批准、重启、file/null origin 处理、白屏 loading 占位和重连循环。
- 优化工作区弹窗底部间距、产物菜单图标渐变色和消息历史完整时间展示。

### Release and CI

- 从 release assets 中排除 builder debug 文件。
- 新增 `.github/release.yml` 自动生成 release notes 分类配置。
- 发布流程支持优先读取 `docs/release-notes/<tag>.md` 作为 GitHub Release 正文，缺失时再退回 GitHub 自动生成说明。
- GitHub Actions 继续产出 macOS、Windows、Linux 的 x64/arm64 安装包，并附带 `SHA256SUMS.txt`。

## Bug Fixes

- 修复 session detail 默认面板、标签唯一性与侧边栏宽度问题。
- 修复 AppImage 环境下 OpenClaw 检测不稳定问题。
- 修复 Gateway 授权、重启、白屏 loading 与重连循环相关问题。
- 修复桌宠状态竞争、持久化、构建、资源路径、拖拽和 IPC 清理问题。
- 修复新产物类型在 parser、node commands 与 HTML 生成流程中的不一致。
- 修复魔法创建弹窗首次不显示、新建弹窗互相干扰和条件渲染问题。
- 修复导航模型、Hub card 类型、scheduled tasks 位置和导航路径 locale 覆盖。
- 修复 Dashboard Gateway 汇总、图表坐标轴和图表容器填充问题。
- 修复 3D office blaster 输入目标、倒下角色控制和场景交互稳定性问题。
- 修复 grouped sidebar nav label 样式、sidebar merge width conflict 和头像 fallback。
