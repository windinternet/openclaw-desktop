# OpenClaw Desktop v0.1.2

## Bug Fixes

- 为左侧边栏顶部用户头像和底部账号头像增加稳定的 fallback 背景，避免头像图片加载失败时显示为空白。

## Tests

- 增加 Sidebar 回归测试，覆盖两个侧边栏头像在图片 URL 失效时仍保留可见 fallback 背景。

## Release

- 继续通过 GitHub Actions 构建 macOS、Windows、Linux 的 x64/arm64 发布产物。
- 发布包附带 `SHA256SUMS.txt` 校验文件。
