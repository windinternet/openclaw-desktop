# 核心理念

## 产品定位

OpenClaw Desktop 是一个面向 AI Agent 时代的**控制台**，连接 OpenClaw Gateway，统一管理：

- Agent 交互（会话、工作流）
- 团队协作（Agent Teams、3D 办公室）
- 系统控制（扩展、记忆、设置）

## 设计原则

### 1. 人类掌舵，智能体执行

- 人类负责：方向决策、验收标准、品味
- 智能体负责：编码、测试、部署、修复
- 人的时间和注意力是唯一稀缺资源

### 2. 仓库即记录系统

- 所有知识必须在代码仓库内（Markdown / 代码 / 模式）
- Google Docs、聊天记录、人脑中的知识都是不可见的
- 智能体在运行时无法访问的任何内容，就是不存在

### 3. 渐进式披露

- AGENTS.md 是地图，不是百科全书（约 100 行）
- 知识在 `docs/` 的子目录中分层存放
- 智能体从小而稳定的入口开始，按需深入

### 4. 智能体可读性优先

- 先为 AI 优化可读性，再为人类优化
- 结构化、可验证、交叉引用
- 定期的"文档养护"扫描过期内容

### 5. 约束即杠杆

- 通过架构约束（分层、依赖方向、边界）来防止漂移
- 不微观管理实现细节
- 品味通过 linter 和检查器编码到系统中
